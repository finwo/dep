__NAME ====== General purpose dependency manager, with a slight focus on static C libraries Usage ----- ``` ``` Installation ------------ To install __NAME, simply download [dist/dep](dist/dep) and place it in your `/usr/local/bin` directory or anywhere else that's included in your `$PATH`. If you want to build it from source, clone the repository and run the following commands: ```sh make sudo make install ``` To install the binary in a location other that `/usr/local/bin`, pass the `DESTDIR` definition to the `make install` command (default: `/usr/local`).
